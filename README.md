# Employee Management System

A full-stack Employee Management System built with React (frontend) and Spring Boot (backend), connected to PostgreSQL.

## Tech Stack
- **Frontend:** React, Vite, Axios
- **Backend:** Java, Spring Boot, Spring Data JPA
- **Database:** PostgreSQL (Render)
- **Deployment:** Vercel (frontend), Render (backend)

## Features
- View all employees
- Add new employee
- Edit employee details
- Delete employee
